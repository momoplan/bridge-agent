# 百积木个人用户安装指南

本文面向第一次使用百积木的个人用户。百积木是本机连接工具，用来把当前电脑安全连接到百积木平台；它本身不是对话客户端。

## 安装

1. 打开公共下载页：

```text
https://baijimu.com/bridge-agent/download
```

2. 下载适合自己电脑系统的安装包：
   - Windows：下载 `.msi`，安装完成后桌面会出现“百积木”图标
   - macOS：下载 `.dmg`
   - Linux：下载 `.deb` 或 `.AppImage`
3. 打开安装包，按系统提示完成安装。
4. 启动百积木。

## 授权

1. 按页面提示登录你的百积木账号。
2. 确认授权当前设备。
3. 授权完成后，百积木会显示已授权或已连接。

## 下一步

百积木当前没有内置对话输入框。授权完成后，在百积木首页点击“打开控制台”，进入百积木控制台后再打开 Agent 对话页面。

如果需要安装 Codex，在百积木平台的 Agent 对话里输入：

```text
请使用 Codex 设备安装能力，给当前工作区的设备安装 Codex。
```

后续按提示继续即可。你不需要手动安装技能，也不需要配置默认助理；平台 Agent 会自动查找并加载 Codex 设备安装能力。平台会先安装并打开 Codex App；如果你的账号或工作区需要 Codex 终端能力，平台会继续自动完成配置。你不需要自己下载 Codex，也不需要复制 API Key。

## 遇到问题

把错误提示或截图发给支持人员。不要发送完整密钥、访问令牌或敏感业务数据。
